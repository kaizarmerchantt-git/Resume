Project 01 / Active

Gas Turbine Engine Simulator

0D Thermodynamic Cycle Analysis Web Application  ·  2026

Python FastAPI Cantera React NASA pyCycle/OpenMDAO Bash Git

Problem

Gas turbine thermodynamic models in educational notebooks had all engine parameters hardcoded. Running parametric studies required editing Python code directly for each iteration, making design space exploration impractical.

Approach

Ported the turbojet cycle model (Flight Test Engineering / GitHub) into a FastAPI Python backend with 12 REST endpoints. Built a self-contained React frontend converting all hardcoded parameters (CPR, TIT limit, efficiencies, areas) into interactive sliders. Integrated the GE CF34-10E turbofan deck from NASA pyCycle/OpenMDAO with trilinear interpolation across altitude, Mach, and power code.

Result

Interactive 0D cycle simulator computing thrust, TSFC, SAR, and gas-path station data at 7 engine stations. T–s diagram generated from Cantera entropy output. Parameter sweeps over altitude, Mach, and throttle with CSV export. CF34-10E turbofan returns 62 performance variables per query in under 50 ms.

Project 02 / Study Project

Thermal Management Simulation

Two-Phase Heat Exchanger Modeling in Simscape  ·  2026

MATLAB Simscape DAE Solvers Thermodynamic EoS Automated Code Generation

Problem

Modeling two-phase heat exchangers for aerospace thermal management requires thermodynamic property data across subcooled, saturated, and superheated regimes. Standard single-phase Simscape libraries do not support this, and manually coding custom components from empirical correlations is error-prone at scale.

Approach

Built a custom multi-layer two-phase heat exchanger interface natively in Simscape. Processed complex equations of state to generate large-scale interpolation grids covering all fluid regimes. Developed MATLAB automation scripts that parse raw thermodynamic data and dynamically synthesize over 1,300 lines of declarative Simscape component code (.ssc), eliminating manual transcription. Translated empirical correlations for flow boiling, pressure drops, and flow regime mapping into purely declarative constraint equations compatible with the DAE solver.

Result

Functional custom simulation components with validated thermodynamic properties (verified against theoretical limits: normal boiling points, critical points). Numerical stability achieved through algebraic loop management, piece-wise property normalization, and smoothed flow reversals, ensuring strict mass, energy, and momentum conservation.

Project 03 / B.E. Thesis

ML in Compressor Blade Design

Axial Compressor Performance Prediction  ·  2022–23

ANSYS BladeGen ANSYS CFX TurboGrid Python KNN Regression Scikit-learn

Problem

Early-stage axial compressor blade design requires repeated high-fidelity CFD evaluations to identify promising geometries, which is computationally expensive during preliminary design screening.

Approach

Parameterised approximately 10–12 axial compressor blade geometries in ANSYS BladeGen varying camber, stagger angle, chord, and blade count. Ran steady-state CFD simulations in ANSYS CFX (meshed in TurboGrid) at multiple RPM and mass flow conditions. Processed simulation datasets with Python and trained a KNN regression model on pressure ratio, temperature ratio, and efficiency outputs.

Result

Compressor performance maps generated across three RPM settings (8,000 / 10,000 / 12,000 RPM) with surge and choke flow identification. KNN model capable of predicting efficiency and pressure ratio trends without full CFD per iteration, enabling faster preliminary design screening.
